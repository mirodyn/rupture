Rupture is structure into two main altering phases with different structure. 
- Weaving 
- Realignment 

Default flow of the phases can be interrupted by Events. When event is triggered by any source, the event is fully resolved according to it's rules, before the phase resumes is default flow. 

# Weaving

Default state of the game is Weaving phase which is devided to rounds during which players take turns in the clockwise order, starting with player with initiative/ balance token. 
During each turn players take precisely one of the following available actions:
- Weave spell
- Add core to spell
- Release the spell
Or, they may decide to take different action made available to them by certain effects. 

When the condititions for the end of phase are met, the current round is finished and then, Realignment phase starts. 

# Realignment 

Realignment phase is fixed order of automated steps that can trigger certain events. If any of the events is triggered, that event is resolved before any other steps are taken during the phase. 
Once all the required steps are taken, Realignment phase ends and new weaving phase begins, starting with the player with initiative token. 

# Events

Events are conditional situations that can occur at any moment of the game and can have various causes. The most common event is Ripple and Events triggered by realignment phase checks. 
Events are always resolved fully before the current phase resumes and can occur during the turn of the player, effectively pausing the turn. 
It's possible for players to apply certain effects during the events based on the effects available to them. In a case when there is conflict between effects of two players during an event, the event of player "closer" to Initiative token gets priority.
